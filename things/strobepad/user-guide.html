StrobePad

user guide

introduction

Thank you for choosing StrobePad.

StrobePad is a compact handheld wireless strobe controller for direct control of a Stairville Power Strobe 1500 DMX via a compatible wireless DMX receiver. It is designed for fast setup, direct operation, and predictable live use.

package contents

  • StrobePad unit
  • USB-C charging cable
  • Quick user guide

device overview

StrobePad provides the following controls and indicators:

  • rotary control 1: flash speed
  • rotary control 2: flash power
  • Fire button: activates live output while pressed
  • integrated display: shows active settings and device status
  • buttons next to the display: used for configuration, test, and calibration functions
  • USB-C port: charging and service power
  • battery compartment: user-replaceable 18650 battery

power and charging

  • StrobePad is powered by a rechargeable 18650 battery
  • charge the device via the USB-C port
  • use only suitable 1-cell Li-ion batteries
  • do not use damaged batteries
  • fully charge the battery before first use
  1. Power on the device.
  2. Verify the displayed DMX start address, wireless ID, and 2.4 GHz wireless channel.
  3. Confirm that the compatible wireless DMX receiver and fixture are powered and configured correctly.
  4. Adjust the flash speed and power controls as required.
  5. Press and hold Fire to activate strobe output.
  6. Release Fire to return output to off.

normal operating behavior

  • output remains off while Fire is not pressed
  • output becomes active only while Fire is held
  • speed and power can be adjusted live while Fire is pressed
  • the display shows active settings and device status during operation

display information

The display provides the following information during normal operation:

  • DMX address
  • wireless ID
  • 2.4 GHz wireless channel
  • device setup and configuration

StrobePad stores the following user settings:

  • DMX address
  • wireless ID
  • 2.4 GHz wireless channel

settings are designed to persist after a power cycle. Configuration is performed using the buttons next to the display. The display shows the currently available button functions.

configuration functions

  • adjust DMX address
  • adjust wireless ID
  • adjuadjust 2.4 GHz wireless channele settings and exit

test mode is intended for functional verification of the device.

test mode is accessed using the buttons next to the display.

test functions

  • potentiometer test
  • button test
  • transceiver test
  • display of current test status and results

Use Test Mode to verify correct input response and wireless hardware operation.

calibration mode is used to calibrate the minimum and maximum travel of both rotary controls.

calibration mode is accessed using the buttons next to the display.

calibration procedure

  1. Start calibration mode.
  2. Move both rotary controls through their full range.
  3. Confirm the calibration values.
  4. Save calibration data.
  5. Exit calibration mode.

Calibration values are stored in the device and remain available after power cycle.

troubleshooting

no output

  • verify that Fire is pressed
  • verify DMX address, wireless ID, and wireless frequency / channel
  • verify receiver and fixture power
  • verify fixture DMX mode and addressing

incorrect response

  • check configured settings on both transmitter and receiver
  • verify fixture channel mapping
  • run Test Mode
  • repeat potentiometer calibration if needed

charging issues

  • verify USB-C cable and power source
  • verify battery condition and polarity

maintenance

  • keep the device clean and dry
  • store at moderate temperature
  • recharge the battery regularly during long storage periods
